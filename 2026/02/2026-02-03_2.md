**AWSのセキュリティ関連サービス**を、役割別に整理したまとめ。

---

## 全体マップ

AWSのセキュリティは、主に以下の6カテゴリに分けて考える。

1. 認証・認可（Identity）
2. ネットワークセキュリティ
3. DDoS / 攻撃対策
4. データ保護（暗号化）
5. 脅威検知・監査
6. コンプライアンス・管理

---

## 1. 認証・認可（Identity）

### IAM（Identity and Access Management）
AWSセキュリティの中核。

- ユーザー / グループ / ロール
- ポリシーによる権限制御
- 最小権限の原則

**試験ポイント**
- EC2 → IAMロールを使用
- アクセスキーの埋め込みはNG

---

### IAMロール
- 一時的な認証情報
- EC2 / Lambda / ECS などで利用

**キーワード**
- 「AWSサービス間アクセス」→ IAMロール

---

### Amazon Cognito
- Web / モバイルアプリのユーザー認証
- ソーシャルログイン対応

**試験キーワード**
- 「アプリ利用者の認証」

---

## 2. ネットワークセキュリティ

### Security Group
- インスタンス単位
- ステートフル
- 許可ルールのみ

---

### Network ACL
- サブネット単位
- ステートレス
- 許可・拒否ルールあり

---

### VPC Endpoint
- AWSサービスへプライベート接続
- インターネット不要

**試験キーワード**
- 「S3をインターネット経由させない」

---

## 3. DDoS / 攻撃対策

### AWS Shield
- DDoS対策
- 全AWSアカウントで自動有効
- L3 / L4攻撃対応

---

### AWS Shield Advanced
- 有料
- 大規模・標的型DDoS対応
- DRT（専門チーム）対応
- コスト保護

**試験キーワード**
- 「ミッションクリティカル」
- 「高度なDDoS対策」

---

### AWS WAF
- L7（HTTP/HTTPS）レイヤの防御
- SQLインジェクション / XSS / HTTP Flood

**連携先**
- CloudFront
- ALB
- API Gateway

---

### Amazon CloudFront
- CDN
- DDoSトラフィック吸収
- Shield / WAFと組み合わせる

---

## 4. データ保護（暗号化）

### AWS KMS（Key Management Service）
- 暗号鍵の作成・管理
- IAMと連携
- CloudTrailで利用ログ記録

---

### CMK（カスタマーマスターキー）
- AWS管理CMK
- カスタマー管理CMK（試験頻出）

**試験ポイント**
- 「厳格な制御」→ カスタマー管理CMK

---

### S3 暗号化
- SSE-S3
- SSE-KMS（最重要）
- SSE-C（ほぼ出ない）

---

### AWS Secrets Manager
- DBパスワードやAPIキー管理
- 自動ローテーション対応

---

### Systems Manager Parameter Store
- 設定値や秘密情報の保存
- 低コスト / 無料枠あり

---

## 5. 脅威検知・監査

### AWS CloudTrail（超重要）
- API操作ログ
- 誰が・いつ・何をしたか

---

### AWS Config
- リソース構成の変更履歴
- 設定違反の検知

---

### Amazon GuardDuty
- 不審な挙動を自動検知
- ログ分析ベース
- 設定が簡単

---

### Amazon Inspector
- EC2 / コンテナの脆弱性診断
- OS・パッケージレベル

---

## 6. コンプライアンス・管理

### AWS Artifact
- 各種コンプライアンスレポート提供
- ISO / SOC / PCI DSS

---

### Amazon Macie
- S3内の機密データ検出
- 個人情報（PII）検知

---

## 7. SAA 即答チートシート

| 要件 | サービス |
|----|----|
| 認証・権限管理 | IAM |
| EC2 → S3 | IAMロール |
| データ暗号化 | KMS |
| HTTPS証明書 | ACM |
| DDoS対策 | Shield |
| HTTP攻撃対策 | WAF |
| 操作ログ | CloudTrail |
| 不審検知 | GuardDuty |
| 設定監査 | Config |
| 機密データ検出 | Macie |

---
