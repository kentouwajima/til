# TIL (Today I Learned)

日々の学習ログを残すためのリポジトリです。  
AWS / インフラ / Linux / Terraform など、**その日学んだことを短く記録**します。

完璧さよりも「継続」を重視します。

---

## 目的

- 学んだことを **自分の言葉で整理する**
- 「分かったつもり」を防ぐ
- 後から見返して **すぐ思い出せる状態**を作る
- 学習の積み上げを可視化する

---

## ルール

- 1日1ファイル（書けない日は無理しない）
- 長文を書かない（5〜10分で終わらせる）
- コピペOK、箇条書きOK
- 他人向けより **未来の自分向け**
